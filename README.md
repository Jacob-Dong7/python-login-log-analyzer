# Python Login Log Analyzer

A project that analyzes login logs and identifies suspicious IP addresses based on failed login attempts.

</div>

---

## Description

This project reads a login log file and checks for unsuccessful login attempts.

If a login attempt fails, the program records the IP address
If the same IP address fails multiple times, the program increases its attempt count
The IP addresses are then sorted from highest to lowest failed attempts and displayed in a risk report

---

## Features

<ul>
  <li>Reads login records from a file</li>
  <li>Detects unsuccessful login attempts</li>
  <li>Counts failed login attempts for each IP address</li>
  <li>Sorts IP addresses from highest to lowest risk</li>
  <li>Uses merge sort to order the results</li>
  <li>Assigns High, Moderate, or Low risk levels</li>
  <li>Prints a clear login analysis report</li>
</ul>
